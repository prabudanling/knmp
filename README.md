## 🚀 **Quick Start**

### **Prerequisites**

```bash
# Node.js 18+
node --version

# Bun (recommended) or npm
bun --version
```

### **Installation**

```bash
# Clone repository
git clone https://github.com/knmp/knmp-website.git

# Masuk ke direktori
cd knmp-website

# Install dependencies
bun install

# Jalankan development server
bun run dev

# Buka browser
# http://localhost:3000
```

### **Environment Variables**

```env
# .env.local
NEXT_PUBLIC_APP_URL=http://localhost:3000
NEXT_PUBLIC_API_URL=https://api.knmp.or.id
MIDTRANS_SERVER_KEY=your-server-key
MIDTRANS_CLIENT_KEY=your-client-key
```

## 📊 **Database Schema**

### **Core Tables**

```prisma
// Users
model User {
  id          String   @id @default(cuid())
  email       String   @unique
  name        String
  phone       String
  tier_id     String
  tier        Tier     @relation(fields: [tier_id], references: [id])
  nik         String?
  photo       String?
  ktp_photo   String?
  farm_photo  String?  // Khusus Petani
  province_id String?
  status      Status   @default(PENDING)
  created_at  DateTime @default(now())
  updated_at  DateTime @updatedAt
}

// Tiers
model Tier {
  id          String   @id @default(cuid())
  name        String
  level       Int
  normalPrice Int
  promoPrice  Int
  benefits    Json
  users       User[]
}

// Positions
model Position {
  id           String   @id @default(cuid())
  level        String   // KORNAS, KORWIL, KORDA, KORCAM, KORDES
  position     String
  name         String?
  status       String   // vacant, filled, pending
  province_id  String?
  regency_id   String?
  district_id  String?
  village_id   String?
  user_id      String?
  created_at   DateTime @default(now())
}
```

## 🌐 **Deployment**

### **Vercel (Recommended)**

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

### **cPanel (Shared Hosting)**

1. Build project: `bun run build`
2. Upload `.next` folder
3. Configure `.htaccess`
4. Set environment variables

### **Docker**

```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]
```
